'Flow-Chart-Maker'

Usage

  • Declare Blocks, e.g
          variable_name => block_type : some_text
  • Connect Blocks, eg.
          variable1->variable2(direction)->condition1
          condition1(yes)->variable3
          condition1(no)->variable4

Block types:

  1. start
  2. end
  3. inputoutput
  4. operation
  5. condition
  6. subroutine
