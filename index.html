'Flow-Chart-Maker'

Usage

  • Declare all blocks as variables, as follows :-
          var_name=>block_type: some_text_you_want , eg:-
                st=>start : Start program
                en=>end: program ends
                cond=>condition: Is x>5 ?
                op=>operation: subtract 3 from x
  • Connect them in proper sequence, eg.
          st->var2->var3(right)->cond
          cond(yes,right)->var2
          cond(no)->en
        You can also add direction of arrow as done for var3 to cond
    if var is condition block, eg cond, there are 2 paths available namely, yes and no

Available types of block

  1. start
  2. end
  3. inputoutput
  4. operation
  5. condition
  6. subroutine
